# Darkness 프로젝트

콘솔 기반 던전 탐험 게임의 전체 소스 루트다. `Program`, `Game`, `GameManager`가 실행 진입점과 게임 상태 전환을 담당하고, `View`와 `Utility`가 콘솔 화면 및 공통 출력 기능을 제공한다.

하위 폴더는 전투, 효과, 월드, UI, 데이터처럼 책임별로 나뉜다. 이 구분은 게임 규칙과 표시 코드를 분리해 기능을 추가하거나 밸런스를 조정할 때 영향 범위를 줄이기 위해 존재한다.

상세한 실행·변경·검증 절차는 [MAINTENANCE.md](MAINTENANCE.md)를 참고한다.
